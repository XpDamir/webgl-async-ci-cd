# webgl-async-ci-cd
Проект

Разработка и настройка CI/CD-пайплайна для WebGL-приложения с асинхронной обработкой игровых сессий.

Цель

Автоматизировать сборку, тестирование и развертывание WebGL-приложения с серверной частью.

Используемые технологии

Unity WebGL / JavaScript

Node.js (сервер)

WebSocket

GitHub Actions

GitHub Pages / Render

Docker (опционально)

Архитектура

Клиент (WebGL) взаимодействует с сервером через WebSocket API.
CI/CD автоматически:

запускает сборку

проверяет код

деплоит клиент

деплоит сервер
